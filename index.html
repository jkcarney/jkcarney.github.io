"Please Don't Do This"

July 2020 - Brothers Studio - GMTK Game Jam

- 2.5D Typing game simulating behavior you definitely shouldn't do

- Created in 48 hours for the Game Maker's Toolkit Game Jam 2020

- Assumed role of debugger/QA programmer. Attempted to break game by testing several edge cases and patching issues that came up.

 - For example: car obstacles were frequently spawning inside each other. I developed an algorithm to ensure that no car would spawn inside another car.

 - Another example: car obstacles had varying driving or scrolling speeds. This meant that cars would glitch into each other often. I troubleshot the issue and developed a fix that stops cars that are about to collide.

- Rated in the top 2% overall from over 5600 global entries.
